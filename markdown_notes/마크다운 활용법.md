# 📝 마크다운(Markdown) 활용법

---

## 1. 마크다운이란?
> 특수기호 몇 개로 텍스트를 꾸미는 간단한 문법  
> .md 확장자 파일로 저장하며, VSCode / Notion / GitHub 등에서 지원

---

## 2. 기본 문법

### 제목
```
# 제목 1
## 제목 2
### 제목 3
```

### 텍스트 강조
```
**굵게**
*기울임*
~~취소선~~
`인라인 코드`
```
결과: **굵게** / *기울임* / ~~취소선~~ / `인라인 코드`

### 목록
```
- 항목 1
- 항목 2
  - 하위 항목

1. 순서 있는 항목
2. 순서 있는 항목
```

### 인용문
```
> 중요한 내용이나 정의를 여기에 씁니다
```
> 중요한 내용이나 정의를 여기에 씁니다

### 구분선
```
---
```

## 3. 코드 블록 (ROS 공부할 때 핵심!)

### 인라인 코드
```
`ros2 run`
```

### 코드 블록 (언어 지정 가능)
````
```python
def callback(self, request, response):
    response.success = request.data  # 요청값을 응답에 담기
    return response
```

```bash
ros2 service list   # 서비스 목록 확인
colcon build        # 빌드 명령어
```
````

## 4. 표 만들기

```
| 항목 | 설명 | 예시 |
|------|------|------|
| Service | 요청-응답 방식 | /set_bool |
| Topic | 지속 발행 방식 | /cmd_vel |
| Action | 장시간 작업 방식 | /move_base |
```

## 8. 파일 저장 위치 (우분투 기준)

```bash
# 마크다운 폴더 생성
mkdir -p ~/ros2_ws/src/markdown_notes

# 새 노트 파일 생성
touch ~/ros2_ws/src/markdown_notes/[파일명].md

# VSCode로 열기
code ~/ros2_ws/src/markdown_notes/[파일명].md
```

---

*이 파일을 `~/ros2_ws/src/markdown_notes/` 에 저장해서 활용하세요! 🤖*
